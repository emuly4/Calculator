Fills the DHCP packet response with the appropriate BOOTP header for DHCP. This fills out all the required fields, leaving only the additional DHCP options to be added to the packet before it is sent to the DHCP server.

Parameters
[out]DHCPHeaderLocation in the packet buffer where the BOOTP header should be written to
[in]DHCPMessageTypeDHCP Message type, such as DHCP_DISCOVER
[in]AppStateApplication state of the current UDP connection
Returns
Size in bytes of the created DHCP packet
