Detailed Description

DHCP Server Application. When connected to the uIP stack, this will send IP configuration settings to a DHCP client on the network.

Fills the DHCP packet response with the appropriate BOOTP header for DHCP. This fills out all the required fields, leaving only the additional DHCP options to be added to the packet before it is sent to the DHCP client.

Parameters
[out]DHCPHeaderLocation in the packet buffer where the BOOTP header should be written to
[in]DHCPMessageTypeDHCP Message type, such as DHCP_DISCOVER
[in]ClientHardwareAddressClient MAC address the created transaction should be directed to
[in]PreferredClientIPPreferred IP that should be given to the client if it is unallocated
[in]TransactionIDTransaction ID the created transaction should be associated with
Returns
Size in bytes of the created DHCP packet
